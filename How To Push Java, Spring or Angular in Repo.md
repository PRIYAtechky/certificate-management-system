How To Push Java, Spring or Angular Codes in GitHub Through GitBash







Step 1: **Download Git Bash from:** 



Step 2: Right Click on the folder you want to upload -> Show more options -> Git Bash



Step 3: Type -> git init



press Enter



Step 4: Type -> git add .



press Enter



Step 5: Type -> git commit -m "Add Customer Service Sprint 1 module"



press Enter



Step 6: Type -> git remote add origin https://github.com/<your-username>/<your-repo>.git (add your repo http url)



press Enter



Step 7: Type -> git branch -M main



press Enter



Step 8: Type ->  git push -u origin main



press Enter



**To Check the Username and Email**



git config user.name -> "Your Name" will appear

git config user.email -> "your\_email@example.com" will appear
